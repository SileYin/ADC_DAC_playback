1. Description

This example utilizes the ADAU1979 ADC and the ADAU1962a DAC to operate in one of two modes:

  • Four channel audio playback (default setting).
  • Four channel analog loopback self test using a 3.0kHz signal.

2. Processor

  • ADSP-SC589

3. Overview

This example demonstrates the usage of the ADAU1979 ADC and ADAU1962a DAC. Both the ADC and DAC are configured to pass 4 channel audio, at a sampling rate of 48kHz, to one/two SPORT devices.

The mode of operation of this example (Playback or Analog Loopback) can be configure in adc_dac_playback.h header file. The #define PLAYBACK is defined by default and builds the example for playback mode. If this #define is commented out the example builds for analog loopback mode. For playback mode there is an additional #define TDM_MODE that determines whether the playback operates in I2S (stereo mode) or multichannel (TDM mode). By default, playback is I2S and consequently the define is commented out as such //#define TDM_MODE. The #define CALLBACK_COUNT determines the playback duration of the example. By default it is set to run for more than 10 seconds.

  • Playback
    Two modes for playback are supported: 1. Stereo Mode: By Default this audio example runs in I2S mode. Connect an audio source to IP1 and IP2 of the four Audio input connectors on J6. Play audio can be heard on OP1 and OP2 of the four (extreme left) Audio output connectors on J16. Connect an audio source to IP3 and IP4 of the four Audio input connectors on J6. Play audio can be heard on OP3 and OP4 of the four (extreme left) Audio output connectors on J16. 2. TDM Mode: This audio example supports TDM4 and TDM8 modes for Playback. To run this audio example in TDM4 mode, uncomment "TDM_MODE" and "TDM4" macros in adc_dac_playback.h file and connect an audio source to IP1 and IP2 of the four Audio input connectors on J6. Play audio can be heard on OP1, OP2, OP3 and OP4 of the four (extreme left) Audio output connectors on J16. To run this audio example in TDM8 mode, comment the TDM4 macro back and uncomment "TDM8" macro in adc_dac_playback.h file, update the macro "NUM_CHANNELS" to "8". In file adc_dac_playback.c, change the second parameter (ADI_ADAU1979_SERIAL_MODE) to (ADI_ADAU1979_SERIAL_MODE_TDM8) in API call adi_adau1979_Open() in function Adau1979Init() and change the second parameter (ADI_ADAU1962A_SERIAL_MODE) to (ADI_ADAU1962A_SERIAL_MODE_TDM8) in API call adi_adau1962a_Open() in function Adau1962aInit(). Connect an audio source to IP1 and IP2 of the four Audio input connectors on J6 and play audio can be heard on all 8 ouput channels.
  • Analog Loopback
    In this mode the example performs an external analog loopback self test on a 3.0 kHz internally generated signal. External loopbacks from all four inputs to all four outputs are required for this example to pass and produce a console output "All Done". A 3.0kHz with a sampling rate of 48kHz signal is generated from SPORT4b. It passes through the ASRC that converts the signal to 96kHz and on to the DAC. The signal is externally looped back to the ADC which passes it on to the ASRC for conversion back to SPORT4a at 48kHz. An autocorrelation is then performed on the received audio to determine that it is a 3.0kHz sine wave.

5. External connections

  • For Playback mode
    Connect an audio source to any of the four Audio input connectors on J6. Play audio can be heard on the corresponding Audio output port of the four (extreme left) connectors on J16.
  • For Analog Loopback
    Connect loopback cables all four audio inputs on the connector J6 to the four (extreme left) Audio output connectors on J16 as shown.
